### ●論文摘要
論文題目為「結合均勻實驗設計與深度學習應用於心電圖之心臟疾病分類」，使用的心電圖<br>
種類分別為正常竇率、心律不整、充血性心臟衰竭，透過收集心電圖數據進行特徵擷取，並<br>
使用均勻實驗設計法結合卷積神經網路AlexNet結構，透過設計AlexNet結構神經網路的超參<br>
數，將訓練及測試資料分為固定及隨機兩種模式，探討是否可達到與Matlab 2019版本之<br>
AlexNet神經網路的辨識結果相當或超越之可能性。
